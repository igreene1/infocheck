# 🛡️ InfoCheck: Threat Intelligence Simulator

**InfoCheck** is a "Serious Game" and digital inoculation tool developed to test the efficacy of gamified interventions against digital misinformation. It was built as part of the AP3134 research module.

## 📌 Project Overview
The spread of fake news is a chronic, daily issue. Traditional "inoculation" studies often rely on passive, text-based learning. This project tests a **longitudinal, active-engagement approach**. 

Users step into the role of a Threat Intelligence Agent. They must review viral social media posts, decrypt independent evidence files, and make a final verdict. The system utilizes **Inoculation Theory** (pre-bunking) to expose users to weakened forms of common misinformation tropes (e.g., Deepfakes, Sensationalism, False Corroboration) and immediately provides corrective feedback.

## 🎮 Features
* **Dual-Condition Testing:** Users can choose between the Gamified Simulation (Experimental) and the Text Training Manual (Control) to directly compare engagement.
* **Dynamic Reputation System:** Choices actively impact a live health bar. Mistakes cost reputation; correct verifications build it.
* **Simulated Rivalry (Agent AI):** The game features an NPC rival (Agent Sterling) who mocks failures and challenges the user, utilizing the *Octalysis Framework* for behavioral engagement.
* **Local Storage Leaderboard:** A persistent, localized leaderboard allows multiple users at an exhibition/kiosk to compete for the highest accuracy score.
* **Immersive UI/UX:** CSS-driven CRT scanlines, text-scrambling decryption animations, and glitching alert notifications.

## 🔬 The Science: Scenarios Included
The simulation targets 6 specific misinformation vectors:
1.  **Public Health Panic** (Lack of Corroboration)
2.  **Tech/Environmental Paranoia** (Correlation vs. Causation)
3.  **Financial Embezzlement** (Converging Evidence)
4.  **Synthetic Media** (Audio Deepfakes & Context Alibis)
5.  **Sensationalism** (Context Dropping/Clickbait)
6.  **Cyber Security** (Phishing & Urgency Tactics)
